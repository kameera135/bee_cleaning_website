# Bee Cleaning Co — Business Website

A single-page business website for a local cleaning service based in Cranbourne, Victoria, Australia.

## Live Site

[beecleaning.netlify.app](https://beecleaningco.netlify.app)

## What's on the page

- Hero section with service highlights
- Services — residential, commercial, end of lease, window cleaning
- How it works — 4-step process
- Service area — Cranbourne and surrounding suburbs
- Customer reviews
- Contact form (sends enquiries directly to business email)

## Tech

Plain HTML, CSS and a small amount of vanilla JavaScript. No frameworks, no dependencies, no build step.

Contact form uses [FormSubmit.co](https://formsubmit.co) to deliver enquiries to the business email without needing a backend.

## How to run locally

Just open `index.html` in any browser. No server needed to view the site.

> Note: the contact form only works when accessed via a live URL, not from a local file.

## Deployment

Hosted on Netlify free plan. To redeploy, drag the updated `index.html` into the Netlify dashboard under **Sites → Deploy manually**.
